# Issue Template for schemas

When creating a GitHub issue for a schema, use this template.
Replace placeholders (`<...>`) with actual content.

## Template

```markdown
<details>
<summary>Full schema</summary>

<Read the schema file and paste its entire content here verbatim.
Do not summarize, truncate, or omit any section.>

</details>
```

## Rules

- The issue **title** should match the schema's `#` heading.
- The issue **body** is the `<details>` block containing the **complete, unmodified** schema content.
- Do not add anything outside this structure.
